Python Basic Data Types

Python的基本数据结构主要包括Tuple、List、Dict、Set。

Tuple

Tuple中的元素一旦创建之后就不能改变，不能增加，不能减少，不能删除。

List

List中的元素创建之后还可以改变，可以增加，可以减少，可以删除。

Dict

Dict用来保存key-value对。其中key必须是immutable的。

Set

Set中的元素是唯一的。

Slice

Tuple和List支持slice。
